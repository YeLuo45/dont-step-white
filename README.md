# 别踩白块（Don't Step on White）

经典钢琴块玩法的 Web + PWA 游戏。

**访问地址**: https://yeluo45.github.io/dont-step-white/

## V3 新功能

### 🪙 金币经济系统
- **获取方式**：
  - 无尽模式通关：min(50, floor(score/5))
  - 关卡通关：20~100金币（根据分数）
  - 每日首次登录：30金币
  - 连续登录3天额外奖励：50金币
- **存储位置**：localStorage `dsw_v3_coins`

### 🛒 皮肤商店
- 4套主题皮肤：
  | 皮肤 | 价格 | 主题色 |
  |------|------|--------|
  | 经典（默认） | 免费 | 深蓝+青色 |
  | 霓虹 | 100🪙 | 深黑+紫色 |
  | 简约 | 80🪙 | 浅灰+深灰 |
  | 赛博 | 150🪙 | 深蓝+青色 |
- CSS变量主题切换（--skin-bg, --skin-accent）
- 存储：`dsw_v3_equipped`（已装备）、`dsw_v3_owned`（拥有列表）

### ⭐ 关卡挑战模式
| 关卡 | 规则 | 通关条件 | 解锁条件 |
|------|------|----------|----------|
| 限时60秒 | 60秒倒计时 | ≥30分 | 默认解锁 |
| 极速模式 | 速度×2 | ≥20分 | 通关限时60秒 |
| 纯净模式 | 无道具 | ≥25分 | 通关限时60秒 |
| 生死局 | 1命+白块从2开始 | ≥15分 | 通关极速模式 |
| 巨慢模式 | 速度×0.5 | ≥40分 | 通关纯净模式 |
| 无限模式 | 无命限制 | 无 | 通关生死局 |

- 星级评价：1星=通关条件、2星=1.5倍、3星=2倍
- 存储：`dsw_v3_progress`（解锁状态、最高分、星级）

### 🎮 UI改动
- **主菜单**：开始游戏/皮肤商店/关卡挑战/设置 + 右上角金币显示
- **设置页面**：音效开关 + 清除数据（慎用）
- **皮肤商店**：顶部金币 + 4卡片网格布局
- **关卡选择**：6关卡 + 锁定状态 + 星级显示
- **游戏结束**：获得金币显示 + 商店/关卡快捷按钮

## V2 功能

### 🏃 无尽模式
- **逐渐增多的白块** — 游戏开始每行1个白块，随分数增加白块数量逐渐增多（1→2→3...，最多3个）
- **命机制** — 初始3条命，踩白块扣1命，漏踩黑块扣1命，3命全失游戏结束
- **指数加速** — 速度随分数指数递增（800ms→180ms下限），每5分加速一次
- **Combo计分** — 连续踩中黑块 combo×2 加分，断则清零

### 🎁 道具系统
| 道具 | 图标 | 效果 |
|------|------|------|
| 护盾 Shield | 🛡️ | 下一次踩中白块不扣命 |
| 冰冻 Freeze | ❄️ | 3秒内所有白块暂停下落 |
| 双倍得分 Double | ✖️2 | 接下来5次踩块得分×2 |

- 30%概率在分数达到10/20/30...时掉落（纯净模式除外）
- PC按 **D键** 使用，移动端**点击图标**使用
- 屏幕顶部显示当前持有道具（最多1个）

### 🎮 UI改动（V2）
- **左上角** — 分数 + Combo连击显示
- **右上角** — 3颗心形生命值
- **游戏结束** — 3字符昵称输入 + 3按钮（重新开始/返回主页/分享）

## 功能特性

- **经典钢琴块玩法** — 4列8行网格，黑块逐行下落
- **操作方式** — PC：← → 移动，空格/点击踩下；移动：左右滑动，点击踩下
- **历史最高分** — localStorage 持久化记录（dsw_v2_best）
- **暂停功能** — P 键暂停/继续
- **音效反馈** — 踩黑块音（800→1200Hz 上升音）、失败音（锯齿波下降音）
- **PWA 支持** — 可安装到 Android/iOS 主屏幕
- **离线缓存** — Service Worker 无网可玩

## 本地运行

```bash
npm install
npm run dev    # 开发模式 http://localhost:5173/dont-step-white/
npm run build # 生产构建
```
